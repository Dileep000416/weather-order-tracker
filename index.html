Weather Order Tracker

This project checks weather conditions for customer order locations using the OpenWeatherMap API and flags delivery delays automatically.

Technologies Used

  • Python
  • asyncio
  • aiohttp
  • HTML
  • CSS
  • JavaScript
  • OpenWeatherMap API

Features

  • Fetches weather data for multiple cities concurrently
  • Flags delayed orders during Rain, Snow, or Extreme weather
  • Handles invalid city errors without crashing
  • Uses environment variables for API security
  • Generates weather-aware apology messages
